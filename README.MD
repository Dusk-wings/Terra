## Overview
**Terra** is a collaborative content management system inspired by platforms like Blogger.  
It allows users to create and manage projects, invite collaborators, and build websites effortlessly.

---

### Tech Stack
- **Frontend:** Next.js, TypeScript, Tailwind CSS  
- **Backend:** Supabase (Auth, Database, Storage)  
- **Deployment:** Vercel  

---

## Features
- Create and manage projects easily  
- Invite collaborators with different access levels  
- Manage content and media through an intuitive interface  
- Built for modern web publishing with real-time updates
